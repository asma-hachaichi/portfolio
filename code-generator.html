Description

This project aims to design and develop a web application containing components and their codes in an appropriate language. This application will automatically generate the skeletons of screens and add the code of its components to new screens through a simple-to-use graphical interface. The objective of this topic is to increase reliability, efficiency, and facilitate complex tasks when writing code for each screen.

Technologies

ReactJS, NodeJS, ExpressJS, PostgreSQL
